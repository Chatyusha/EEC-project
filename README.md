# How to Use

`$ mkdir <target-directory>`, (for example, `$ mkdir ~/Library`),
`$ cd <target-directory>`, and `$ git clone https://github.com/Chatyusha/Shell.git Shell`

Replace the first line of zshrc with 'export LIBRARY=\<target-library\>'
